# ReverseSearchAPI

Api de Scrap para realizar busca reversa no serviço do Google imagens

POST /reverseSearch
Adicionar na Url Query a url

# Get Started

Clonar o projeto. Na linha de comando, vá até a pasta do projeto e rode
`npm i`
`npm run nodemon`

O projeto ira rodar na posta 3000
